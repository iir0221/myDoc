* 因为上下文切换（CPU切换执行线程，需要保存前一个任务的状态，这都将影响效率）
* 通过无锁并发编程（例如借助于Hash算法取膜分段，不同线程处理不同数据），CAS算法（Atomic包使用了CAS算法），使用最少线程（比如说设置servlet container的线程池中的线程数为一个合理的值）和协程（nodejs）
