# 整数集合，为了节省空间
## 作用
* set key的底层实现之一
    * 当一个set只包含整数值元素并且集合的元素数量不多时。
## 结构
```c
typedef struct intset {
    
    // 编码方式
    uint32_t encoding;

    // 集合包含的元素数量
    uint32_t length;

    // 保存元素的数组,数值按照从小到大排列
    int8_t contents[];

} intset;
```

## 升级
* 新元素类型比现有所有元素类型都长时
* 提升灵活性
* 节约内存

## 不支持降级

## 时间复杂度
* 由索引获取key，O(1)
* 增加，删除O(n)(因为需要升级)
* 查找O(logn)(有序数组，二分)
